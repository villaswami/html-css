What's the Difference?

HTML

HyperText markup language

    create the structure
  • controls the layout of the content
  • provides structure fro the web pages design
  • the fundamental building block of any web pages

CSS

cascading style sheet

    Stylize the website
  • Applies style to the web pages elements
  • target various screen sizes to make web pages responsive
  • primarily handles the 'look and feel' of the web page

javascript

    increase interactivity
  • applies handels interactivity
  • handles complex functions and features
  • programmatic code which enhances functionality
